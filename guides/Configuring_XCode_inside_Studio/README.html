Configuring XCode inside Studio

Switch between Xcode 4.3 and other Xcode versions (Xcode 4.2 and lower)

Studio utilizes the xcode-select (typically found in /usr/bin/xcode-select) script to retrieve the desired Xcode installation. If you have multiple Xcode installations and need to switch what version Studio uses, you need to do the following.

  1. Open a Terminal window and execute the command "sudo xcode-select -switch <xcode_folder_path>". The <xcode_folder_path> is typically /Developer or /Developer-x.x for Xcode 4.2 and lower, and /Applications/Xcode.app/Contents/Developer for Xcode 4.3

  2. In Studio, open the Studio Preferences and navigate to Studio >Platforms > iOS

  3. In the iOS section, click the Refresh button

  4. The newly specified Xcode folder path should be displayed.

You may need to restart Studio for your changes to take effect.
