Changing the Update Type

Note, you may already have these updates sites configured, and you just need to enable or disable them as appropriate.

  1. Select Preferences:

    • On Windows, Preferences is found under the Window menu.

    • On Mac, Preferences is found under the Aptana Studio 3/Titanium Studio (or Eclipse on Eclipse) menu.

  2. In Preferences, expand the Install/Update arrow and select Available Software Sites.

  3. Click the Add button, and one of the sites below to get the particular update stream. Name it something similar to "Studio XYZ Updates":

  4. Save your changes, and exit the preferences dialog.

  5. Go to the Help menu, and select Check for Titanium Updates or Check for Appcelerator Updates. A dialog will appear once a new update is detected, and you could then follow the instructions to download the update.

Available Updates

  • Stable releases are those with the most testing, and the same as you would download from the website.

  • Beta releases are those with some testing, and are in preparation to be released relatively soon.

  • Nightly updates are bleeding-edge releases. This is straight from the developer server. As such, you will see new features first, but there is the possibility of blocker bugs or file corruption. Use at your own risk.
