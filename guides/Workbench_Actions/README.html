Requirements

The examples in this page use the Prototype library, which is included by default inside a portal. However, the same concepts may be applied using other implementations.

Invocation

This command is executed immediately in a synchronous way.

Get the Active Perspective Information

To get information regarding the active perspective in the Studio, dispatch this command:

var activePerspective = dispatch($H({
controller : 'portal.workbench',
action : "getActivePerspective"
}).toJSON());

The activePerspective variable will hold a Map that contains the following keys: label, description and id.
