Appendices

Legacy Command-Line Interface

This appendix covers how to use the command-line interface to the titanium script provided by the Mobile SDK. You can use this command line interface to perform various common tasks such as creating, building, and running Titanium Mobile projects and modules.

tiapp.xml and timodule.xml Reference

The tiapp.xml and it's close cousin the timodule.xml are XML configuration files that Titanium Mobile uses to generate native resources and code. This appendix covers the various options you can set within those configuration files.

ECMA-262-5 Compliance

This appendix explores the features of JavaScript you can use within your Titanium Mobile applications by listing the various platforms' compliance with the ECMA 262-5 specification.

Mobile Web Limitations

This appendix describes some of the inherent limitations and challenges of the Mobile Web platform.
