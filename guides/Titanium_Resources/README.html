Titanium Resources

Objective

Appcelerator strives to make the developer experience as effective and efficient as possible. In this section, you will explore some of the various resources available to Titanium Developers. These resources include development tools, documentation, example code, and multiple sources for developer support.

Contents

Developer resources

  • The my.appcelerator.com site is your portal for Titanium developer information. You can download Titanium builds, access documentation and resources, and much more via your "My Appcelerator" page.

  • The Developer Center is your source for developer news, the Developer Blog, the DevLink developer directory, and much more.

Documentation resources

  • Official Titanium Docs

Example code

You'll find sample code, example projects, app components, and more at the following github repositories.

  • Appcelerator github repository – Titanium source code, free add-on modules, and the KitchenSink

  • Appcelerator Titans repository – Sample apps and components from the Titans

  • Appcelerator Services – Sample apps and code for Appcelerator customers (log into http://support.appcelerator.com to request access)

  • Personal repositories of Appcelerator staff and community members

    • Kevin Whinnery – Helium testing framework, Suds SOAP helper, examples

    • Dawson Toth – Titanium Redux framework, sample apps

    • Pedro Enrique – App components, examples

    • Ben Bahrenburg (Community member) – Add on libraries, samples, components, plus check out his list of helpful links for Titanium developers

    • Matt Apperson

    • Terry Martin (Community member) – App components and samples

(Apologies for anyone we left off. This list is hardly exhaustive.)

Summary

In this section, you explored some of the resources available to Titanium Developers, including developer tools, documentation, example code, and multiple sources of developer support. To make full use of these, though, you'll need to gain a level of competency with the Javascript language. In the final section of this chapter you'll get a crash course in Javascript development, giving you the knowledge necessary to build your first mobile app with Titanium.
