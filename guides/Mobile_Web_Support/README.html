Mobile Web Development Prerequisite

Mobile Web SDK

Titanium Mobile SDK version 2.0 or newer is recommended for Mobile Web development.

Creating a Mobile Web Project

Open the new Mobile Project wizard and select 'Mobile Web' as a deploy-target.

Previewing a Mobile Web Project in an External Browser

Once you have created a mobile web app, you will be able to preview your app in a external web browser. To preview a mobile web project in a browser, open the project in either the App Explorer or Project Explorer, click on the launch button, and select "Mobile Web Preview in Browser".

The launch will create a run configuration that can be edited under the "Run Configurations..." dialog. In the run configurations, you will be able to customize the browser you want to use to preview your project.

Previewing a Mobile Web Project in an Emulator

This is currently only working for the Android Emulator. If you do not have a valid Android SDK installed and configured, you will not be able to preview your mobile web project in an emulator.

To preview your app in an emulator, open the project in the App Explorer, or Project Explorer, click on the launch button, and select "Mobile Web Preview in Emulator".

Packaging a Mobile Web Project

To package a mobile web project, simply click on the launch button and select "Package - Mobile Web".
