Node.ACS

Appcelerator Custom Cloud Service (Node.ACS) allows users to publish Node.js based services into the cloud. This integration builds on top of the command-line interface.

For information about managing Node.ACS projects in Studio, see the Node.ACS Studio guides.

For information about getting started with Node.ACS services, see the Node.ACS Quick Start guide.
