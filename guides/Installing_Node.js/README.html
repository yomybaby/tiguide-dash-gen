Troubleshooting

If you run into issues installing or configuring Node.js, below are some common solutions.

NPM not installed

Studio installs a version of NPM (the Node package Manager) by default, but if you installed Node yourself, you might not have done so already. To do so, do:

curl https://npmjs.org/install.sh | sh

On the command line.
