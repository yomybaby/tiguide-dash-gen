Viewing your Installation History

  1. Go to About Studio.

  • In Studio

    • On Windows, go to Help and select About Studio

    • On Mac, go to Studio and select About Studio

  • In Eclipse

    • On Windows, go to Help and select About Eclipse SDK

    • On Mac, go to Eclipse and select About Eclipse SDK

  1. Select Installation Details.

  2. Select the Installation History Tab.

  3. Select the time you are interested in. It will show you the versions of the plugins installed at that time.

Reverting to a Previous Build of Studio

Although Studio does provide this feature, it is not generally recommended and can leave your Studio in an unstable state.

Due to a limitation in Eclipse, you will only be able to revert to the last stable version. Examples:

Possible: Revert from 1.0.8 (beta) to 1.0.7 (stable)
Not possible: Revert from 1.0.7 (stable) to 1.0.6 (stable)

If you wish to revert to an earlier stable version, you will need to reinstall using a link below.

If you would like to revert your current version of Studio to a previous build, you can revert to a previous build using the About Studio window.

  1. Follow the steps above to view the installation history

  2. Select the time you would like to revert to and then click Revert.
