Overview

This guide explains the steps for modifying your configuration file.

Modifying the INI Configuration File

  1. Locate the INI configuration file:

    1. Mac OS X:

      1. Navigate to the /Applications/Aptana Studio 3, /Applications/Titanium Studio or /Applications/Appcelerator Studio folder in Finder.

      2. Right-click (or control-click) on AptanaStudio3.app, TitaniumStudio.app or AppceleratorStudio.app and choose Show package contents.

      3. In the new Finder window navigate to Contents/MacOS.

    2. Linux: Navigate to the location of your Studio application (it is wherever you unzipped the download)

    3. Windows: Navigate to the location of your Studio application (likely under C:\Users\username\AppData\Local\Aptana Studio, C:\Users\username\AppData\Local\Titanium Studio or C:\Users\username\AppData\Local\Appcelerator Studio)

  2. Open AptanaStudio3.ini, TitaniumStudio.ini or AppceleratorStudio.ini

  3. Edit the options as desired

  4. Save and close the file

You can follow the same procedures on Eclipse. The folder structure will be identical but the file to edit is eclipse.ini.
