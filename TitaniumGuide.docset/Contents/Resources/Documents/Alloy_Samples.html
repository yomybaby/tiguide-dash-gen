Studio 3.2.0 and later includes a number of Alloy samples and the Alloy Github project contains many smaller test examples. Below is a brief overview of the samples available.

Sample Applications

Importing a Sample

You can import any of these samples from the Studio start page or from the Samples view.

To import a sample from the start page:

  1. If the Dashboard is not open, in the menu bar, select Help > Studio > Show Studio Dashboard.

  2. Click on the Develop tab on the dashboard.

  3. Find the sample you want to import and click the Import button.

  4. Follow the wizard prompts to finish importing the project.

To import a sample from the Samples view:

  1. If the Samples view is not open, in the menu bar, select Window > Show View > Samples.

  2. Right-click on a project and click Import sample as project..., or select a project and click the import button ( ).

  3. Follow the wizard prompts to finish importing the project.

Test Case Examples

Importing an Example

For more simple usage examples of Alloy, go to the alloy/test/app directory on the Alloy GitHub project.

To use the sample applications from the GitHub project, first clone the project:

git clone https://github.com/appcelerator/alloy

Using Studio

  1. Create a new Alloy project.

  2. Select File > Import.

  3. Select General > File System and click the Next button. The Import dialog appears.

  4. Click the first Browse button (From directory), then navigate to a test application in the cloned Alloy project and click the Open button. For example, if you want to import the Master-Detail application, select alloy/test/apps/advanced/master_detail/.

  5. Select all folder and files except the specs if it exists in the test app.

  6. Click the second Browse button (Into folder), then navigate to your project's app directory and click the OK button.

  7. If the test app had a specs folder, open the app/controller/index.js file and remove the require('specs/index')($) call.

  8. Click the Finish button. The sample application is imported in your project and you can run it.

Using the CLI

Run the following commands:

ti create --name ProjectName
cd ProjectName
alloy new
cp -r <path_to_alloy_git_repo>/test/apps/<path_to_app>/* app/.
## If the test app contains a specs folder,
## Open the app/controller/index.js file
## Remove the require('specs/index')($) call.
rm -rf app/specs
ti build
