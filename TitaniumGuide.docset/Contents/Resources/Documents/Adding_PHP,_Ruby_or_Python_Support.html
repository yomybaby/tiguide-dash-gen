Adding PHP, Ruby or Python Support

As of Studio Release 3.1.0, support for server-side languages is optional to slim down the Studio installer. To add them back in again:

  1. Launch Studio

  2. Go to Help > Install New Software…

  3. Enter the following URL in the Work with: field:

    • PHP - http://download.appcelerator.com/aptana/studio3/php/update/stable

    • Ruby - http://download.appcelerator.com/aptana/studio3/ruby/update/stable

    • Pydev - http://download.appcelerator.com/aptana/studio3/pydev/update/stable

  4. Check the top level node in the table and go through the wizard to install.
