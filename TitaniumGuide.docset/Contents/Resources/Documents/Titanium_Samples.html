Importing a Sample

You can import any of these samples from the Studio Dashboard or from the Samples view.

To import a sample from the dashboard:

  1. If the Dashboard isn't open, click Help > Studio > Show Studio Dashboard.

  2. Click on the Develop tab on the dashboard.

  3. Find the sample you want to import and click the Import button.

  4. Follow the wizard prompts to finish importing the project.

To import a sample from the Samples view:

  1. If the Samples view isn't open, click Window > Show View > Samples.

  2. Right-click on a project and click Import sample as project, or select a project and click the import button ( ).

  3. Follow the wizard prompts to finish importing the project.

Kitchen Sink

The Kitchen Sink samples are demonstrations of the feature set for Titanium Mobile. While examples of what the platform can do, they are not meant to be used as a starting point for application development. However, you can check out Commands > Titanium Mobile to find a suite of snippets designed to extract the most useful pieces from the Kitchen Sink projects for use in your own applications.

Geolocation

This is a Titanium sample app that uses native maps to plot locations. With it you can forward geocode addresses and add them as annotations to the map.

Topics Covered

  • Using native maps with Titanium.Map

  • Adding annotations to native maps

  • Remote Data access via Titanium.Network.HTTPClient

  • Modular Javascript with CommonJS

  • Cross-platform design

  • Documentation Links

Git repo: https://github.com/appcelerator-developer-relations/Sample.Mapping

RSS Reader

This is a Titanium Mobile sample app that creates a RSS reader. With it you can pull live RSS feeds from the internet, list them along with thumbnails, then drill down to the article itself.

Topics Covered

  • Remote Data acccess via Titanium.Network.HTTPClient

  • Modular Javascript with CommonJS

  • Multiple window app using a single execution context

  • Loose coupling of UI and data code using application level events via Ti.App

  • Native UI features

    • Android menus

    • iOS navigation bar buttons

    • iOS Navigation Controller via Titanium.UI.iPhone.NavigationGroup

  • Webviews for displaying web content

  • Cross-platform design

Git repo: https://github.com/appcelerator-developer-relations/Sample.RSS

Todo List

  • Multiple window app using a single execution context

  • Android menus

  • iOS navigation bar buttons

  • Cross-platform design
